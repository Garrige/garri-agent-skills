# Skill: lead_capture

## Purpose
Help design a contact form and lead-capture flow.

## When to use
If user asks about:
- contact form fields
- what to ask the client
- what happens after form submit
- integration with email/CRM

## Recommended contact form fields
Minimum:
- Name
- Email or Phone
- Message
Optional (only if needed):
- Budget range
- Preferred contact channel
- Deadline

## After-submit flow (simple)
1) Show “Thanks” message on site
2) Send email notification to owner
3) Optional: save lead to Google Sheet / CRM

## Output
Return:
- fields list
- validation notes
- short “Thanks” message text
