# Skill: faq_writer

## Purpose
Generate FAQ blocks and short website copy.

## When to use
When user asks to write:
- homepage text
- services description
- FAQ / Q&A
- short CTA text

## Steps
1) Identify target audience and offer in 1 sentence.
2) Write 5–8 FAQs:
   - question in simple language
   - answer 1–3 sentences
3) Add 3 short CTAs (different tones):
   - neutral
   - direct
   - friendly

## Output format
- Title
- 5–8 FAQ items (Q/A)
- 3 CTAs
